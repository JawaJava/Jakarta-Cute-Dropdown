Enjoy.... 

# Support Browser
* Internet Explorer 7+
* Firefox 4+
* Safari 3.1+
* Google Chrome 4+
* Opera 12+

# Requirement
* jQuery 1x(for ie < 8) or 2x
* [ie8.js](http://ie7-js.googlecode.com/svn/version/2.1(beta4)/IE8.js) or [page](https://code.google.com/p/ie7-js/) for compatible ie 7, 8
